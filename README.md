# Tensile-Machine-Data-Processing-and-Plotting
Plot the Strain‑Stress scatter plot and the Young’s modulus‑fracture strain double bar chart for mechanical testing

**Instructions for Use:**

1. During testing, calculate stress and strain in the tensile testing machine. The fourth column should be strain, and the fifth column stress. Export the data as a CSV file. For different samples, name the first one as `[Sample Name]_1`.
2. For the sample with the best test results, add a `_b` suffix to its CSV file name. Move the files that need to be plotted to a custom folder. Modify the image output path to your desired location.
3. Run the code in MATLAB and select the folder containing the files.
